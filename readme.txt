■概要

  これは「伺か」互換ベースウェア「SSP」で動作するゴーストです。
  動作には.NET Framework 2.0以上が必要となります。

  「里々」のログビューア「れしば.NET」のゴースト版です。
  普通に「れしば.NET」単体で起動したほうが便利だと思います。

  れしば.NET
    https://github.com/nikolat/recv_net
